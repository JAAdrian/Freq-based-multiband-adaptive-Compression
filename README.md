# Frequency-based Multi-Band Adaptive Compressor for Hearing Aides

This Python package implements the algorithm published in [1] with proper testing and
convenient interfaces.

## Reference

PATEL, Kashyap; PANAHI, Issa MS. Frequency-based multi-band adaptive compression for
hearing aid application. In: Proceedings of Meetings on Acoustics. Acoustical Society of
America, 2019. S. 055004.

[1]: https://doi.org/10.1121/2.0001247
